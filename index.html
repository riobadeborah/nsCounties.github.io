Nova Scotia Counties API Documentation

This API provides information about counties. You can use the following endpoints:

Endpoints:

  • GET /counties: Get information about all counties.
  • GET /counties/:name: Get details of a specific county by name.
  • POST /counties/:name: Update the details of a specific county by name.

Example:

Retrieve information about all counties:

GET /counties

Retrieve details of a specific county (e.g., Halifax):

GET /counties/Halifax

Update details of a specific county (e.g., Halifax):

POST /counties/Halifax
Content-Type: application/json

{
  "name": "Halifax",
  "population": 480,523,
  "coordinates": {
    "latitude": 44.6488,
    "longitude": -63.5752
  }
}

Note: Replace "Halifax" with the name of the desired county.
